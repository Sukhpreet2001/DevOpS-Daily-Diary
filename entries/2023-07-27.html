Jenkins Setup

Setting Up Jenkins Using Docker

Follow these steps to set up Jenkins using Docker:

Step 1: Run Docker Container

Open PowerShell as an administrator and execute the following command to run the Jenkins container:

docker run -p 8080:8080 -p 50000:50000 -d -v jenkins_home:/var/jenkins_home jenkins/jenkins:lts

Step 2: Check Container Status

Use the following command to check the status of the Docker container and obtain the container ID:

docker ps

Step 3: View Container Logs

Retrieve the password token for Jenkins from the container logs using the container ID:

docker logs container_id

Step 4: Access Jenkins Web Interface

Open a web browser and go to localhost:8080.

Step 5: Log In to Jenkins

Enter the password token obtained from the logs and proceed to log in to Jenkins.

Follow these steps to successfully set up Jenkins using Docker and begin automating your software development processes.
