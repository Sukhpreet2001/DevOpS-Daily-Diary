Learning about Dockerfile

Machine -> Intermediate (Github) -> Target (Test/Production)

Once Docker Desktop is installed, follow the steps below to create an image on Docker Desktop:

Step 1: Create a Folder

Create a new folder on your local machine where you'll store your Docker project.

Step 2: Open in VSCode Editor

Open the folder in your Visual Studio Code (VSCode) editor to manage your Docker project.

Step 3: Create a File named app.js

In the folder, create a new file named app.js.

Step 4: Write Inside the File

Write the following line inside the app.js file:

Console.log("Hi, Welcome to Docker!");

Step 5: Open Terminal and Execute the Command

Open the terminal within VSCode and execute the following command:

node app.js

Note: If Node.js is not previously installed, install it before running the command.

Create a New File Named Dockerfile

Create a new file in the same folder named Dockerfile. Write the following content in this file:

FROM node:alpine
COPY ./app.js /app
WORKDIR /app
CMD node app.js

Tip: The content of the Dockerfile may vary depending on the programming language and requirements of your project.

Save and Build the Docker Image

Save the Dockerfile and open the terminal in VSCode. Execute the following commands:

docker --version
docker build -t hello-docker .

After running the above commands, the Docker image will be created.

Run the Docker Container

Execute the following command to run the Docker container:

docker run hello-docker

The message written in the app.js file will be displayed as the output!

Task 1: Push the Image to Docker Hub (hub.docker.com)

Follow the steps below to push this image from your local system to Docker Hub:

  1. Tag the Image:
    2. docker image tag hello-docker:latest docker_username/hello-docker:latest
  2. Login to Docker Hub:
    3. docker login
  3. Push the Image:
    4. docker image push docker_username/hello-docker:latest

Alternatively, you can use the following steps:

  1. Login to Docker Hub:
    2. docker login
  2. Tag the Image:
    3. docker tag hello-docker:latest docker_username/repository_name:latest
  3. Push the Image:
    4. docker push docker_username/repository_name:latest

Task 2: Test Using Play-with-docker

Follow the steps below to test the Docker image using Play-with-docker:

  1. Open Play-with-docker:

    2. Open the Play-with-docker website to access the Docker playground.

  2. Add a New Instance:

    3. Add a new instance, and an editor will open.

  3. Run the Container:

    4. Execute the following command to pull and run the Docker image:

    docker run docker_username/hello-docker:latest

    The message written in the app.js file will be displayed!
