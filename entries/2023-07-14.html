Day 4 - 14/07/2023 Friday

Development Workflow Steps

  1. Create a folder named hello_docker.
  2. Open the folder in VS Code.
  3. Create a file named app.js and write the following code:
    4. console.log("Hi, Welcome to Docker");

    Terminal Command: node app.js

    It will print: Hi, Welcome to Docker

  4. Create a new file named Dockerfile in VS Code. Install any required extension.

    5. Dockerfile Content:

    FROM node:alpine
COPY . /app
WORKDIR /app
CMD node app.js

    Save the Dockerfile and check the version of Docker.

  5. Use the command docker build -t hello_docker . and press enter to build the image.
  6. Use the command docker image ls to see the list of images created.
  7. Use the command docker run hello_docker to check if the Docker image is working.

Task 1: Push the File to Docker Hub

Tips:

  • Use the command docker login to log in to Docker Hub.
  • Tag the image using docker tag hello_docker:latest kaurcs2001/devops_training_2001:latest (Replace kaurcs2001 with your Docker Hub username).
  • Use the command docker push kaurcs2001/devops_training_2001:latest to push the image to Docker Hub.

Task 2: Test the Pushed File

Website: playwithdocker (node will not work, so clone Docker and try running the image).

Commands Used:

  1. docker
  2. docker pull kaurcs2001/devops_training_2001
  3. docker run kaurcs2001/devops_training_2001
